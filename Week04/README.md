# 学习笔记

### 字符串分析算法

#### 字典树
+ 大量高重复字符串的存储与分析
  用来精确的匹配字符串，检查两个字符串是否完全匹配；
  比如说处理1亿个字符串，存在多少出现频率前50数量的字符串，应该怎么处理；
  比如搜索关键词；

#### KMP
+ 在长字符串里找模式
  检查长字符串是否存在短字符串的部分匹配；

#### Wildcard
+ 带通配符的字符串模式
  问号：表示匹配任意字符；
  星号：表示匹配任意数量的任意字符；
  比如文件查找
  贪心算法

#### 正则
+ 字符串通用模式匹配

#### 状态机
+ 通用的字符串分析

#### LL LR
+ 字符串多层级结构分析
